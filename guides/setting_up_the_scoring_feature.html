Here's another guide to use one of Inkxbot's features, this is a quick guide so it's not that long.
So you came to this page to learn how to set up the scoring feature for your team right? Well let's get down to buisness.



First create a text channel on your server called #scrim-scores, make sure Inkxbot has permissions to send messages to that channel.

Finaly you'll need to write your team's name to Inkxbot's database with ,wt "YOUR TEAM'S NAME HERE"

You're all set! The scoring feature is set up!< br /> Now since it's ready for scores of scrims, you can just log them with ,post scrim {your teams score} "the opposing team's name" {the opposing teams score}
